# Android应用拦截器

一个功能强大的Android应用，帮助用户管理和控制设备上应用的使用。支持黑名单和白名单两种模式，可以有效拦截不希望使用的应用。

## 功能特点

- 🎯 **双模式拦截**：支持黑名单（拦截选定应用）和白名单（仅允许选定应用）模式
- 🔒 **权限管理**：自动检查和请求必需的系统权限
- 📱 **应用管理**：浏览和选择设备上安装的应用
- ⚙️ **灵活设置**：可配置保护开关、自动启动和通知选项
- 🛡️ **安全保护**：支持密码保护功能
- 📊 **实时状态**：显示权限状态和应用拦截状态

## 权限要求

应用需要以下权限才能正常工作：

- **悬浮窗权限**：用于显示拦截界面
- **使用情况访问权限**：监控应用使用情况
- **电池优化豁免权限**：确保后台服务正常运行

## 安装说明

1. 下载APK文件到Android设备
2. 在设置中允许"未知来源"应用安装
3. 点击APK文件进行安装
4. 启动应用并授予所有必需权限

## 使用方法

### 基本设置

1. **启动应用**：首次启动会引导您设置密码
2. **授予权限**：点击"请求必需权限"按钮，授予所有系统权限
3. **选择模式**：在主界面选择黑名单或白名单模式

### 管理应用列表

1. 点击"管理应用列表"按钮
2. 浏览设备上安装的应用
3. 选择要拦截或允许的应用
4. 点击"保存选择"完成设置

### 配置选项

在设置界面可以配置：
- **启用保护**：开启/关闭应用拦截功能
- **自动启动**：开机自动启动保护服务
- **显示通知**：拦截时显示通知提醒

## 系统要求

- Android 5.0 (API Level 21) 或更高版本
- 最少 50MB 可用存储空间
- 支持所有主流Android设备

## 故障排除

### 权限问题
如果遇到权限相关错误：
1. 进入手机设置 → 应用管理 → 应用拦截器
2. 检查所有权限是否已授予
3. 重新启动应用

### 应用无法拦截
1. 确认无障碍服务已启用
2. 检查电池优化是否已禁用
3. 确认应用正在后台运行

### 闪退问题
1. 点击错误界面的"复制错误信息"按钮
2. 将错误信息发送给开发者
3. 尝试重启应用或重新安装
